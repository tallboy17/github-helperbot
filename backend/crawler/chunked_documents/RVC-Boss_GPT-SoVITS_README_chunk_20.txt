The TTS annotation .list file format:  
```

vocal_path|speaker_name|language|text

```  
Language dictionary:  
- 'zh': Chinese
- 'ja': Japanese
- 'en': English
- 'ko': Korean
- 'yue': Cantonese  
Example:  
```

D:\GPT-SoVITS\xxx/xxx.wav|xxx|en|I like playing Genshin.

```
